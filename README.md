# Converf

Converf is a comprehensive Flutter-based mobile application that serves as a modern marketplace and project management platform tailored for contractors. 

## Key Features

* **Marketplace & Bidding:** Contractors can browse marketplace projects, view project details, manage their bidding preferences, and submit detailed multi-step proposals (including introductions, milestone breakdowns, timelines, and agreements).
* **Project Management:** Once onboarded to a project, contractors gain access to project settings and comprehensive management tools such as task lists, document sharing, team management, and milestone tracking.
* **Financials & Billing:** The app provides robust functionality for contractors to manage their billing, view financial summaries, and handle invoices seamlessly.
* **Profile & Onboarding:** Features a structured, user-friendly onboarding flow for new contractors, alongside dedicated profile management and help/support screens.

## Tech Stack

- **Framework:** Flutter
- **State Management:** Riverpod
- **Routing:** GoRouter
- **Assets Support:** SVG & standard image assets

## Getting Started

### Prerequisites
- Flutter SDK (^3.11.0)
- Dart SDK

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Visino2/converf.git
   ```

2. Navigate to the project directory:
   ```bash
   cd converf
   ```

3. Install dependencies:
   ```bash
   flutter pub get
   ```

4. Run the application:
   ```bash
   flutter run
   ```
